Time Ordered Data¶

class toast.tod.Streams(mpicomm=<mpi4py.MPI.Intracomm object>, timedist=True, detectors=None, flavors=None, samples=0)[source]¶

Base class for an object that provides a collection of streams.

Each Streams class has one or more detectors, and each detctor might have different flavors of data and flags.

DEFAULT_FLAVOR¶

string

the name of the default flavor which always exists.

Parameters:
  • mpicomm (mpi4py.MPI.Comm) – the MPI communicator over which the data is distributed.
  • timedist (bool) – if True, the data is distributed by time, otherwise by detector.
  • detectors (list) – list of names to use for the detectors.
  • flavors (list) – list of EXTRA flavors to use (beyond the default).
  • samples (int) – pre-initialize the storage with this number of samples.

class toast.tod.Pointing(mpicomm=<mpi4py.MPI.Intracomm object>, timedist=True, detectors=None, samples=0)[source]¶

Base class for an object that provides detector pointing for an observation.

Each Pointing class has one or more detectors, and this class provides pointing quaternions and flags for each detector.

Parameters:

  • timedist (bool) – if True, the data is distributed by time, otherwise by detector.
